About This System

ML-Based System for Predicting College Student Academic Performance

Project Overview

A frontend-only online system demonstrating machine learning principles for predicting student academic performance using modern web technologies.

Features

  • ML-powered prediction system
  • Backend API with Flask framework
  • Real-time results and analytics
  • 12 input features (academic + behavioral)
  • Visual feature contribution analysis
  • Fully responsive design

How It Works

  1. Enter your unique student details
  2. Provide academic performance data
  3. Submit for ML analysis
  4. View predicted marks and grade

Input Features

📚 Academic Features (8)

  • Attendance: Percentage of classes attended (0-100%)
  • Study Hours: Daily self-study hours (0-12)
  • Previous CGPA: Last semester CGPA (0-10)
  • Internal Marks: Internal assessment (0-20)
  • Mid-Semester Marks: Midsem exam score (0-30)
  • Backlogs: Number of failed subjects (0-5+)
  • Assignment Completion: Submission rate (0-100%)
  • Class Participation: Engagement level (1-5)

👤 Demographic & Behavioral (4)

  • Age: Student age (17-30)
  • Gender: Male/Female/Other
  • Extra-Curricular: Non-academic activities participation
  • Late-Night Entry: Hostel entry frequency (1-5)

Output & Grading System

ML Output

Predicted End-Semester Marks: 0-50 (generated by machine learning model)

Total Marks Calculation

Total = Mid-Semester (20) + Internal (30) + End-Semester (50) = 100

Grade Scale (Rule-Based)

O 90-100 (Outstanding)
E 80-89 (Excellent)
A 70-79 (Very Good)
B 60-69 (Good)
C 50-59 (Average)
D 40-49 (Below Average)
F Below 40 (Fail)

Note: Grade is computed using fixed academic rules, NOT predicted by ML
